 🎮 Hangman Game (Python)

📌 Overview

This is a simple command-line Hangman game developed in Python. The player has to guess a randomly selected word one letter at a time. The game ends when the player correctly guesses the word or runs out of attempts.

 ✨ Features

* Randomly selects a word from a predefined list.
* Allows one-letter guesses.
* Displays the current progress using underscores.
* Tracks guessed letters.
* Prevents duplicate guesses.
* Validates user input.
* Gives the player 6 attempts to guess the word.
* Displays win or lose messages.

 🛠️ Technologies Used

* Python 3
* `random` module
